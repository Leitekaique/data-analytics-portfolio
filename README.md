# Análise Estratégica do E-commerce Olist

Este projeto analisa dados públicos do maior e-commerce brasileiro (Olist) para responder perguntas estratégicas nas áreas de **Pricing, Logística e Customer Experience**.

## 🎯 Principais Insights

- **Top 3 categorias** (beleza, relógios, cama/mesa/banho) concentram 40% da receita total
- **Oportunidade de pricing**: produtos de ferramentas/jardim têm preço 50% abaixo da média da categoria, mas alto volume de vendas
- **Gargalo logístico**: diferença de 20 dias no prazo de entrega entre SP e RR
- **Impacto do atraso**: nota cai de 4,29 para 2,57 quando o pedido atrasa
- **Sazonalidade**: pico de vendas em novembro (Black Friday) e janeiro

## 🛠️ Ferramentas utilizadas

- **SQL** (SQLite) - extração e tratamento dos dados
- **Python** (Pandas, Matplotlib, Seaborn) - visualizações
- **Google Colab** - desenvolvimento e documentação
